# Dataset

This project uses the **Chest X-Ray Images (Pneumonia)** dataset provided by Kaggle.

## Dataset Source
- Kaggle: Chest X-Ray Images (Pneumonia)
- Link: https://www.kaggle.com/paultimothymooney/chest-xray-pneumonia

## Description
The dataset contains chest X-ray images labeled into two classes:
- **NORMAL** – healthy lungs
- **PNEUMONIA** – lungs affected by pneumonia

The images are already organized into training, validation, and testing sets.

## Expected Folder Structure

After downloading and extracting the dataset, organize it as follows:

Place the `chest_xray` folder inside this `dataset/` directory.

## Notes
- Do **NOT** upload the dataset to GitHub due to size limitations.
- Files such as `.DS_Store` or `_MACOSX` can be safely deleted.
